# Musical-Sheeps

  Minecraft plugin running with Spigot 1.8.8. (NMS version: 1_8_R3)
  
  Musical Sheeps is a minigame based on the Musical Chairs IRL game.
  
  Features:
    Multi-Arena
    Free
    Possibility to translate
    Easy to use
    and more.
  
  Dependencies:
    NoteBlockAPI*

    I give the source code so everybody can have a custom version for oneself.
